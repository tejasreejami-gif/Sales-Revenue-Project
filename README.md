Sales & Revenue Analysis Dashboard

📌 Project Overview
The Sales & Revenue Analysis Dashboard is a Power BI project designed to analyze sales performance, revenue trends, product performance, and regional sales distribution. The dashboard provides interactive visualizations that help businesses make data-driven decisions.

🎯 Objectives
Analyze sales and revenue data.
Identify top-performing products.
Compare sales across different regions.
Monitor revenue trends over time.
Provide interactive filtering for better insights.
